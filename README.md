# RollCall-Attendance-Management-System-
🚀 Features

🎓 Teacher Portal

Mark Attendance: Teachers can mark daily or class-based attendance.

View Attendance Reports: Teachers can view detailed attendance records for each student.

CSV Export: Export attendance data in CSV format for easy record-keeping and sharing.

Generate PDF Reports: Teachers can generate and download a PDF of the attendance report for a student or class.

Manage Students: Add, edit, and remove students from the class roster.

Search & Filter: Search and filter attendance records by date, student, or attendance status.

👨‍👩‍👧 Parent Portal

View Child’s Attendance: Parents can see their child’s attendance history, including absences and late arrivals.

CSV Export: Parents can download the attendance history in CSV format.

PDF Report: Parents can generate a PDF attendance report for their child, making it easy to keep records.

Notifications: Parents are notified about missed classes or attendance irregularities.

🧑‍🎓 Student Portal

View Own Attendance: Students can view their own attendance status for each class.

Real-Time Updates: Students can see attendance data updated by teachers in real-time.

PDF Report: Students can generate and download their own attendance report in PDF format.

🛠️ Technologies Used

HTML: Structure of the application and portals.

CSS: Styling, with the use of Tailwind CSS for utility-first, responsive design.

JavaScript: Adds interactivity, such as attendance marking, report generation, and form validation.

Tailwind CSS: For rapid, responsive design using utility-first class names.

CSV: For exporting attendance data in CSV format.

jsPDF: For generating PDF reports of student attendance.

Local Storage/Database (Optional): Store attendance data for persistence.
